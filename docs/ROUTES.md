# Rotas principais

## Páginas
- `/` (Home)
- `/transactions`
- `/debts`
- `/ai`
- `/reports`
- `/settings`

## Transações
- `GET /transactions`
- `POST /transactions`
- `PUT /transactions/{transaction}`
- `DELETE /transactions/{transaction}`

## Dívidas
- `GET /debts`
- `POST /debts`
- `PUT /debts/{debt}`
- `DELETE /debts/{debt}`

## Assistente IA
- `GET /ai`
- `POST /ai/generate`
- `POST /ai/confirm`
- `POST /ai/feedback`

## Views salvas
- `POST /views`
- `PUT /views/{view}`
- `DELETE /views/{view}`

## Administração (Hotmart)
- `GET /admin/hotmart`
- `POST /admin/hotmart`
