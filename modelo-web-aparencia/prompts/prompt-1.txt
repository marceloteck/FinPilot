# PROMPT 1/6 — FinPilot (Laravel + Inertia + Vue) — App Shell + Tema + Acessibilidade + Período Global

Você é um engenheiro de software sênior. Trabalhe em um projeto Laravel já existente com Inertia.js + Vue 3 (estrutura já pronta). Implemente a base visual e estrutural do app (App Shell), com suporte completo a tema e acessibilidade, e um seletor global de período (Ano/Mês) que será usado por todo o site.

## Objetivo desta etapa
1) Implementar o layout global (desktop + mobile) com:
   - Sidebar no desktop
   - Bottom navigation no mobile
   - Topbar com título, subtítulo, seletor global de Ano/Mês e botões de ação
2) Implementar tema (Claro/Escuro/Sistema) e acessibilidade (Fonte: normal/grande/extra; Contraste: normal/alto) usando:
   - atributos no elemento `<html>`: data-theme, data-font, data-contrast
   - persistência em localStorage (chave: "finpilot_prefs_v1")
   - JS utilitário independente (sem dependências externas)
3) Criar base de rotas/páginas (apenas placeholders agora), para navegação:
   - Home (Dashboard)
   - Transações
   - Dívidas
   - IA
   - Metas
   - Relatórios
   - Configurações
4) Implementar seletor global de Ano/Mês (apenas UI + estado persistido por enquanto):
   - Ano ativo (obrigatório)
   - Mês ativo (opcional, default "Todos")
   - Botão "Hoje" volta para ano/mês atuais
   - Persistir ano/mês em localStorage (pode ser na mesma chave de preferências ou outra)
   - Expor ano/mês para todas as páginas via Inertia shared props (ex: use page.props)

⚠️ IMPORTANTE
- Não remover nada existente. Adicionar de forma organizada.
- O layout deve ser responsivo e elegante, “meio termo” (nem escuro demais nem branco demais).
- Implementar suporte completo ao CSS do design que já definimos (usar como base o CSS abaixo).
- Manter o bloco .list-item exatamente como está no CSS (ver abaixo).

## Tema + Acessibilidade (JS)
Crie resources/js/finpilot/theme.js
- Lógica: preferências (theme/font/contrast) com defaults:
  - theme: "system"
  - font: "md"
  - contrast: "normal"
- Persistir em localStorage: "finpilot_prefs_v1"
- Aplicar em document.documentElement:
  - data-theme = theme
  - data-font = font
  - data-contrast = contrast
- Deve “auto iniciar” ao carregar o app.

Além disso, oferecer suporte a UI com selects (se existirem):
- #themeSelect (system/light/dark)
- #fontSelect (md/lg/xl)
- #contrastSelect (normal/high)
Se não existirem, não quebrar.

Implementar atalho opcional:
- tecla "T" alterna theme: system -> light -> dark

## Período global (Ano/Mês) (JS)
Crie resources/js/finpilot/period.js
- Guardar:
  - active_year (number)
  - active_month (string: "all" | "01".."12") ou algo equivalente
- Defaults:
  - active_year = ano atual
  - active_month = "all"
- Persistir em localStorage (pode ser chave separada "finpilot_period_v1")
- Expor funções:
  - getPeriod()
  - setYear(year)
  - setMonth(month)
  - resetToToday()
- Emitir evento custom no window quando mudar (ex: "finpilot:period-changed") para as páginas atualizarem se necessário.
